# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this project,
please report it privately to **security@fireblocks.com**.

Please do **not** open a public GitHub issue. We will acknowledge receipt
within two business days and work with you on a coordinated disclosure
timeline.

## Scope

This repository contains the x402 facilitator service. Reports about the
underlying Fireblocks platform should follow the [Fireblocks security
disclosure process](https://www.fireblocks.com/responsible-disclosure/).
